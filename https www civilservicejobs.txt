HOW TO INSTALL REACT

Check if you already have Node:

node -v
npm -v

If you see versions → good.

If not, install Node LTS from:
👉 https://nodejs.org

(Download LTS version)

Step 2 — Create React App with Vite

In your project folder:

npm create vite@latest frontend

It will ask:

Project name → frontend

Framework → React

Variant → JavaScript (or TypeScript if you prefer)

Then:

cd frontend
npm install
Step 3 — Start Development Server
npm run dev
